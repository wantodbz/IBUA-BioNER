# IBUA-BioNER

This repository contains the implementation of **IBUA-BioNER**, a
framework for few-shot Biomedical Named Entity Recognition (BioNER) that
combines Improved Balanced Undersampling (iBUS) and Adaptive
Mention-Replacement (ADAMER).

The repository includes the source code, datasets, grid search results,
and generated outputs used in the experiments.

## Repository Structure

``` text
IBUA-BioNER/
│
├── code/               # Jupyter notebooks for experiments
├── Dataset/            # BioNER datasets
├── gridsearch/         # Grid search results
├── outputs_ncbi/       # Outputs for the NCBI Disease dataset
├── outputs_bc5cdr/     # Outputs for the BC5CDR dataset
├── outputs_bc2gm/      # Outputs for the BC2GM dataset
└── README.md
```

## Folder Description

### `code/`

Contains all Jupyter notebooks used in the experiments.

-   `final-ibua-ncbi-full.ipynb` : Main experiment on the NCBI Disease
    dataset.
-   `final-ibua-bc5cdr-part.ipynb` : Main experiment on the BC5CDR
    dataset.
-   `final-ibua-bc2gm-part.ipynb` : Main experiment on the BC2GM
    dataset.
-   `GridSearch_K_NCBI.ipynb` : Grid search for parameter **K**.
-   `GridSearch_RS_NCBI.ipynb` : Grid search for random seed selection.

### `Dataset/`

Contains the datasets used in the experiments.

-   NCBI Disease
-   BC5CDR
-   BC2GM

### `gridsearch/`

Stores the CSV files containing grid search results.

### `outputs_ncbi/`, `outputs_bc5cdr/`, `outputs_bc2gm/`

Each output folder contains:

-   `split_indices_*.json` : Few-shot train/validation split indices.
-   `ibus_filter_log_*.csv` : Logs generated during iBUS filtering.
-   `adamer_aug_output_*.csv` : Augmented samples generated by ADAMER.
-   `seed_details_*.csv` : Detailed results for each experimental seed.

## Running the Experiments

Open the notebook corresponding to the desired dataset in the `code/`
directory and execute all cells.
